To set up and run this project:

Install Required Software:

VS Code from https://code.visualstudio.com/
Create Project:


mkdir atm-app
cd atm-app
npm create vite@latest . -- --template react-ts
Install Dependencies:

npm install
Run the Application:

npm run dev
